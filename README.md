# graper
A Spider FrameWork.

## Installation
You can install graper by simply doing:

    pip install graper
    
## Usage
**Very Important Thing**
This line must be the first line in code
```
from graper.spiders import *
```


## About Environment Variable
> ~/.graper/.env < $(pwd)/.env
